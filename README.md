## Groq AI Finance Assistant

The voice finance assistant uses browser speech recognition for free voice capture, then sends the captured text to Groq for structured financial analysis.

Each user saves their own Groq API key from [console.groq.com/keys](https://console.groq.com/keys) inside their account card in the app. The key is stored encrypted on the server.

Optional server settings in `.env.local`:

```env
GROQ_FINANCE_MODEL=openai/gpt-oss-20b
USER_SECRET_KEY=replace_this_with_a_long_random_secret
```
